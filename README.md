# YAMLCord

YAMLCord is an easy-to-use syntax for creating custom actions for Discord bots using YAML.

## üöÄ Example

```yml
sequences:
  - if: "[message.content] eq '!hi'"
    then:
      - create_message: "Hello!"
      - add_reaction: "üëã"
```
